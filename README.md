# dApp Airdrop

This repository demonstrates how to create an Airdrop DApp from A to Z using Wagmi V2, Viem V2, RainbowKit V2, Hardhat, NextJS, and TypeScript.

### Frameworks and Libraries Used:
- #####  Wagmi V2
- #####  Viem V2
- #####  RainbowKit V2
- #####  Hardhat
- #####  NextJS
- #####  TypeScript
- #####  Chakra UI
